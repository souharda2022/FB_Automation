
Facebook Comment Scraper Chrome Extension

How to install it:

1. Open Chrome
2. Go to chrome://extensions/
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select this folder

How to use:

1. Open a Facebook post
2. Scroll to load all comments
3. Click extension icon
4. Click "Scrape & Download CSV"
5. CSV file will download automatically

Notes:
- Facebook UI changes may break selectors
- Scroll fully to load all comments before scraping
